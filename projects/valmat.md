As a private equity intern, I initiated and led a project using Excel programming to allow new formula-based analyses of past employee projects.  Programmatically removing frequent, varied discrepancies in naming conventions and making Excel button interfaces to automate complex retrieval, placement, and formatting of string sets from closed workbooks helped eliminate some of the manual data manipulation required by employees.

I found Excel Visual Basic to share a fair deal of the syntax of object-oriented languages with which I was more farmiliar, and it was a good learning experience in searching documentation and online resources to learn a new set of functionalities and syntax.
