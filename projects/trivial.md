I began this project earlier this year while playing trivia with friends remotely.  We had been running our trivia sessions manually, and sometimes, through websites such as Kahoot, but we wanted some custom functionalities.

Most importantly, we wanted a way for the trivia "host" to be able to manually grade responses while the participants were answering the following question, and for the results to display before the next question was shown.

I have been implementing this project in Visual Studio Code primarily using Node.js, Socket.io, and React JS.  I am still in the early stages of the project, but I hope to having a running version by the end of the year!
