Given Javascript’s widespread use today, I find learning it to be quite worthwhile.  Many professionals and hobbyists make use of a plethora of Javascript frameworks available, many of which have large communities and regular updates, such as node.js and React.  These libraries and support resources help bridge the gap between newcomer and professionally capable Javascript developer, as well.

I have limited past experience with Javascript, but concepts from other object-oriented languages that I’ve used more make the syntax feel somewhat familiar.  The lack of type declarations for variables and return types, contrasting gives a nice “ease of use” element to Javascript, contrasting C++ and Java, in which I have moderate past experience.  However, this lack of explicit types may lead to unforeseen and/or hard to locate errors in larger-scale systems, so I will try my best to be careful in that way.
